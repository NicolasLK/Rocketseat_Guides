## Síncrono

* Sistema síncrono (synchronous), uma tarefa é concluida após a outra.
[ ] A tarefa anteriror precisa ser concluída, para então iniciar a proxíma.
[ ] Por padrão, o JavaScript é um sistema síncrono.
